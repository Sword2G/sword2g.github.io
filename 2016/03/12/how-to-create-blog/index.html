Github + Hexo 构建博客

关于HexoHexodoc官方文档

hexo是一个基于Node.js的静态博客程序，可以方便的生成静态网页托管在github。

前期准备

1 安装Node.js下载地址；

2 安装git 任意位置打开git命令行，输入npm install -g hexo；

3 安装可以编辑markdown的编辑器（如sublime）。

安装Hexo并本地预览

1 在任意路径打开git bash，然后执行npm install -g hexo；

2 在你喜欢的位置新建一个文件夹（比如我F:\hexo），并cd进去执行hexo init初始化；

3 新建一个页面’hexo new testpage’；

4 执行hexo server启动本地服务器；

5 打开浏览器，输入localhost:4000出现网页既证明Hexo安装成功。

使用Hexo写文章

1 执行`hexo new “postName”（postName为文章的名字，一会儿编辑文章的时候也可以更改） ；

2 到目录hexo\source_posts\下找到postName.md文件，用sublime编辑文章 ；

3 执行hexo generate生成静态文件 ；

4 执行hexo server启动本地服务器预览效果 。

hexo常用命令

hexo new ==> hexo n
hexo generate ==> hexo g
hexo server ==> hexo s
hexo deploy ==> hexo d

关联到github

1 登录自己的guihub账号新建一个仓库，格式为*yourname*.github.io(格式必须这样)

2 修改_config.yml文件
deploy:
type: git
repository: git@github.com:Sword2G/sword2g.github.io.git #你自己的仓库
branch: master

3 部署到github
执行hexo deploy完成部署。

4 打开浏览器输入yourname.github.io查看,ok。
